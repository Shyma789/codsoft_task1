Titanic Survival Prediction using Python and Machine Learning.  This project analyzes the Titanic dataset and predicts passenger survival using classification techniques. # 🚢 Titanic Survival Prediction

## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning techniques.  
It also performs data analysis and visualization to identify important factors affecting survival.

---

## 🎯 Objectives
- Analyze the Titanic dataset
- Perform data preprocessing and cleaning
- Build a classification model to predict survival
- Visualize key patterns using graphs

---

## 📊 Features
- Data cleaning and handling missing values
- Survival prediction using classification
- Visual representation of data using graphs
- Insightful analysis of survival patterns

---

## 📈 Visualizations Included
- Survival Count Distribution
- Survival by Gender
- Survival by Passenger Class
- Age Distribution by Survival
- Correlation Heatmap

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📁 Dataset
- Titanic Dataset (CSV file)
- Contains passenger details like age, gender, class, fare, etc.
