# CodSoft Data Science Internship - Completed Tasks

This repository contains the complete set of projects developed during my Data Science internship at CodSoft. Each project focuses on different aspects of machine learning, including classification, regression, and advanced data visualization.

---

## 📂 Project Summaries

### 🚢 Task 1: Titanic Survival Prediction
* **Objective:** Predict whether a passenger survived the Titanic disaster based on features like age, gender, and ticket class.
* **Key Techniques:** Data Cleaning, Feature Engineering, and Logistic Regression/Random Forest.
* **Visualization:** Analysis of survival rates across different demographics.

### 🎬 Task 2: Movie Rating Prediction
* **Objective:** Build a model that estimates the historical rating of a movie based on its genre, director, and actors.
* **Key Techniques:** Handling categorical data, Regression analysis, and handling missing values in large datasets.
* **Visualization:** Trend analysis of movie ratings over various years.

### 🌸 Task 3: Iris Flower Classification
* **Objective:** Classify iris flowers into three species (Setosa, Versicolor, and Virginica) based on petal and sepal measurements.
* **Key Techniques:** Random Forest Classifier and Standard Scaling.
* **Interactive Features:** A high-end **Integrated Performance Dashboard** featuring:
    * **3D Species Cluster Map**
    * **AI Confusion Matrix Heatmap**
    * **Feature Influence (Bar Chart)**
    * **Detailed Performance Metrics Table**
